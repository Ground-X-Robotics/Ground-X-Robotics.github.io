How to join our Minecraft Server on Xbox

Our Minecraft server supports cross-platform play with Bedrock and Java! The current version of the server is 1.16.5, so make sure you are running that version!


Here's how to do it:

  1. Open up your 'Settings' in Xbox
  2. Go to Network Settings
  3. Go to Advanced settings
  4. Go to DNS settings
  5. Press Manual
  6. For the Primary IPv4 DNS, enter:
    104.238.130.180
  7. Then, for the Secondary IPv4 DNS, enter:
    8.8.8.8

  8. Now load Minecraft. Reload it if it was already loaded.
  9. Go to the Servers tab, and join any server.
  10. Press Connect to a Server
  11. Enter the IP Address as mc.groundx.uk
  12. Enter the Server Port as 25616
  13. Check Add server to list
  14. Press Submit


You should now be in the server! If you ever want to join back, the Ground-X Minecraft server should appear in the server list whenever you open the secret server menu again!

Didn't work?
Make sure that you entered the IPv4 addresses into the Network Settings correctly, and that the server IP and Port was typed into Minecraft correctly.
Also assure that you are running the same version of the game as the server.
Right now, the server is running 1.16.5, but will be updated to 1.17 as soon as the Purpur and Crossplay versions are released!
