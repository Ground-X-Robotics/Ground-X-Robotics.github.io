Minecraft Server

The Ground-X Minecraft server!

We decided that a great way to open up our arms and increase our reach, would be to launch a Minecraft server! Of course, heavily moderated. This server is the culmination of work over weeks, and we're happy to release it to the public!


Our server supports cross play! That means that all versions of the game can play together! Java and Bedrock alike. For information on how to join on Xbox, see this page for help!

You can join the server at mc.ground.uk!

Java:
Address: mc.groundx.uk

Bedrock:
Address: mc.groundx.uk
Port: "25616", "25620" or "28018".

Features

This server is constantly in development, but as of right now, there are 2 Worlds with infinite possibilities!
At the moment, there is a Survival world, and a Plots world. With more to come!


Here's what we've got! :

  • Casual Survival
  • TPA
  • /echest,/craft,...
  • Essentials!
  • Land claim
  • Kits
  • Shops (/buy,/sell)
  • Anti-Xray
  • Nightskip
  • Toggleable PVP
  • RTP
  • Plots Squared
  • ...And More!...




The plots world is a fully fledged Plots experience, with an infinitely generating grid of plots for you to claim and build on!
